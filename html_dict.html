TAGS HTML UTILIZADAS

Este dicionário contém as principais tags HTML utilizadas na construção das páginas deste site.

<!DOCTYPE html>
Define o tipo de documento como HTML5. É a primeira linha em um documento HTML.
<html>
É o elemento raiz de uma página HTML. Todos os outros elementos devem ser descendentes deste elemento.
<html lang="pt-br">...</html>
<head>
Contém metadados/informações para o documento, como título, links para CSS, meta tags, etc.
<meta>
Define metadados sobre um documento HTML, como charset, viewport, descrição, etc.
<meta charset="UTF-8">
<title>
Define o título do documento, que aparece na aba do navegador.
<title>Gean Pereira - Página Pessoal</title>
<link>
Define a relação entre o documento atual e um recurso externo, como folhas de estilo CSS ou ícones.
<link rel="stylesheet" href="styles.css">
<body>
Contém o conteúdo visível do documento HTML.
<header>
Representa o cabeçalho de uma seção ou página. Geralmente contém elementos como logo, título e navegação.
<nav>
Define uma seção de navegação com links para outras páginas ou partes da página atual.
<main>
Especifica o conteúdo principal de um documento. Deve ser único no documento.
<section>
Define uma seção em um documento, como capítulos, cabeçalhos, rodapés ou qualquer outra seção do documento.
<aside>
Define um conteúdo relacionado ao conteúdo principal, mas que pode ser considerado separado dele, como uma barra lateral.
<footer>
Define o rodapé de uma página ou seção. Geralmente contém informações de autoria, direitos autorais, contato, etc.
<address>
Define informações de contato para o autor/proprietário de um documento ou artigo.
<h1> a <h6>
Define cabeçalhos HTML. <h1> é o mais importante e <h6> o menos importante.
<h1>Bem vindo!</h1>
<p>
Define um parágrafo de texto.
<a>
Define um hiperlink, usado para vincular de uma página a outra ou a uma seção dentro da mesma página.
<a href="https://github.com/geanpereira" target="_blank">GitHub</a>
<img>
Define uma imagem no documento HTML.
<img src="foto.jpg" alt="Descrição da imagem">
<div>
Define uma divisão ou seção em um documento HTML. É um container genérico para conteúdo de fluxo.
<span>
Define uma seção em um documento. É um container inline genérico para conteúdo fraseado.
<ul>
Define uma lista não ordenada.
<li>
Define um item de lista em listas ordenadas ou não ordenadas.
<style>
Contém informações de estilo (CSS) para um documento ou parte de um documento.
